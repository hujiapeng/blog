1. 管道和信号一样，是进程通信的方式之一
2. 管道符"|"，将前一个命令执行的结果传递给后面的命令，如下
```
[root@ittutu ~]# ps | cat
  PID TTY          TIME CMD
 3795 pts/1    00:00:00 bash
 7151 pts/1    00:00:00 ps
 7152 pts/1    00:00:00 cat
```
3. 管道符是通过开启一个子进程来执行命令的，所以像cd pwd这种内建命令是无法在管道符表达式中正确执行的，如果需要这些内建命令，就要把这些命令封装到脚本中执行。如下切换目录，pwd仍然是root目录
```
[root@ittutu ~]# cd /tmp | pwd   
/root
```
